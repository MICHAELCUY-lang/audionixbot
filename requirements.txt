# Panduan Instalasi Bot Telegram Konversi Media & Pencari Lagu

## Prasyarat
- Python 3.8 atau lebih baru
- Akun Telegram
- Server atau komputer untuk menjalankan bot (lokal atau hosting)

## Langkah-langkah Instalasi

### 1. Membuat Bot di Telegram
1. Buka aplikasi Telegram dan cari "@BotFather"
2. Mulai percakapan dan ketik `/newbot`
3. Ikuti instruksi untuk memberi nama bot Anda
4. Setelah selesai, BotFather akan memberikan token API bot
5. Salin token tersebut untuk digunakan pada langkah berikutnya

### 2. Menyiapkan Lingkungan Pengembangan
1. Buat folder baru untuk proyek bot Anda
2. Buat file Python baru (misalnya `bot.py`) dan salin kode yang telah disediakan
3. Buat folder bernama `temp_files` di lokasi yang sama dengan file Python
4. Siapkan gambar bernama `static_image.jpg` untuk digunakan saat konversi MP3 ke MP4

### 3. Instalasi Dependensi
Buka terminal/command prompt dan jalankan:

```bash
pip install python-telegram-bot yt-dlp moviepy youtube-search-python requests
```

### 4. Konfigurasi Bot
1. Buka file `bot.py` yang telah dibuat
2. Ganti nilai `TOKEN = "YOUR_TELEGRAM_BOT_TOKEN"` dengan token yang Anda dapatkan dari BotFather

### 5. Menjalankan Bot
1. Pada terminal/command prompt, navigasikan ke folder proyek Anda
2. Jalankan bot dengan perintah:
   ```bash
   python bot.py
   ```
3. Bot sekarang aktif dan siap digunakan

## Penggunaan Bot

### Perintah yang Tersedia
- `/start` - Memulai bot dan menampilkan pesan selamat datang
- `/help` - Menampilkan bantuan penggunaan
- `/mp3tomp4` - Memulai proses konversi MP3 ke MP4
- `/mp4tomp3` - Memulai proses konversi MP4 ke MP3
- `/music [nama lagu]` - Mencari dan mengunduh lagu

### Cara Menggunakan Fitur Konversi
1. Kirim perintah `/mp3tomp4` atau `/mp4tomp3`
2. Kirim file sesuai format yang diminta
3. Bot akan mengunduh file, melakukan konversi, dan mengirimkan hasilnya

### Cara Mencari dan Mengunduh Lagu
1. Kirim perintah `/music [nama lagu]` atau ketik `cari lagu [nama lagu]`
2. Bot akan menampilkan beberapa hasil pencarian
3. Pilih lagu yang diinginkan dengan mengklik tombol
4. Bot akan mengunduh dan mengirimkan file lagu dalam format MP3

## Konsiderasi Penting

### Legalitas
Bot ini mengambil konten dari YouTube dan platform lain. Pastikan penggunaan bot ini sesuai dengan hukum yang berlaku di wilayah Anda dan hanya untuk penggunaan pribadi.

### Batas Ukuran File
Telegram memiliki batas ukuran file 50MB untuk bot standar. Untuk file yang lebih besar, Anda perlu mempertimbangkan solusi alternatif seperti mengunggah ke cloud storage.

### Performa dan Sumber Daya
Proses konversi file membutuhkan sumber daya komputasi. Pastikan server atau komputer Anda memiliki spesifikasi yang memadai, terutama untuk file berukuran besar.

## Pemecahan Masalah

### Bot Tidak Merespons
- Periksa apakah bot masih berjalan di server
- Pastikan token API yang digunakan sudah benar

### Error Saat Konversi
- Pastikan semua dependensi terinstal dengan benar
- Periksa apakah format file yang dikirim sesuai dengan operasi yang dipilih
- Verifikasi bahwa file `static_image.jpg` tersedia (untuk konversi MP3 ke MP4)

### Masalah Pencarian Musik
- Periksa koneksi internet
- Verifikasi bahwa library `youtube-search-python` dan `yt-dlp` terinstal dengan benar

## Pengembangan Lebih Lanjut
Bot ini masih dapat dikembangkan dengan fitur-fitur tambahan seperti:
- Konversi format media lainnya (WAV, FLAC, dll.)
- Pemilihan gambar kustom untuk konversi MP3 ke MP4
- Pengaturan kualitas output
- Fitur pemotong audio/video
- Fitur playlist untuk pengunduhan musik
